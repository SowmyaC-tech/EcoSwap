# EcoSwap

## Project Overview
EcoSwap is a web-based platform that allows users to exchange reusable items instead of throwing them away. The main goal is to reduce waste, encourage recycling, and promote sustainable living.

## Features
- User Registration and Login
- Add Items for Swap
- Search Available Items
- Request Item Swaps
- Manage Swap Requests
- User Profile Management
- Notifications

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP
- Git & GitHub

## How to Run
1. Install XAMPP.
2. Copy the project folder into `htdocs`.
3. Start Apache and MySQL.
4. Import `ecoswap.sql` into phpMyAdmin.
5. Open:
   http://localhost/ecoswap/

## Database
The database file is included as:
`ecoswap.sql`
